# React + Vite

Esta aplicação foi desenvolvida utilizando Vite e React

## Arquitetura e Técnicas utilizadas

A aplicação é organizada em uma estrutura de separação de responsabilidades, unindo todos os dados em um objeto - em molde de um json - manipulável e renderizado por estados através de um componente de contexto. Cada parte da composição da página foi desenvolvida de forma única e unida posteriormente no componente App, como blocos de construção. A aplicação utiliza o framework Tailwind para estilização, fazendo uso de CSS grid e Flexbox na organização dos itens em cada elemento.
